# Generated API Client

> **Do not edit this folder manually.** All files here are auto-generated by NSwag from the backend's OpenAPI spec.

## How to regenerate

1. Make sure the API is running at `http://localhost:5000`
2. From `apps/web/`, run:

```bash
npm run generate:api
```

This reads `apps/web/nswag.json`, fetches `http://localhost:5000/swagger/v1/swagger.json`, and writes `client.ts` into this directory.

## What gets generated

- `client.ts` — a fully-typed `JobulerApiClient` class (and sub-clients per controller) plus TypeScript interfaces for every DTO exposed by the backend.

## Requirements

- The .NET API must be running at `http://localhost:5000` before running the generator.
- `nswag` devDependency must be installed (`npm install` from `apps/web/`).
