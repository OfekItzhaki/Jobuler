## How to run

```bash
# Start services
docker compose -f infra/compose/docker-compose.yml up -d

# Frontend dev
cd apps/web && npm run dev

# API
cd apps/api && dotnet run --project Jobuler.Api/Jobuler.Api.csproj

# E2E tests (requires both services running)
cd apps/web && npx playwright test --reporter=list

# TypeScript check
cd apps/web && npx tsc --noEmit

# Frontend build
cd apps/web && npx next build

# API build
dotnet build apps/api/Jobuler.Api/Jobuler.Api.csproj
```
